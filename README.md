시한부인생 프리채널 아케이드챤의 위치를 나타내주는 지도입니다.

시즌3 시작하고 어째서인지 고장나는 기계들이 많아 다른 지역의 챤 위치를 알고싶을때가 있었는데, 공홈에 나온 지역명만 보고서는 위치파악이 힘들었습니다.
